stratify :: Ord b => (a -> [b]) -> [a] -> [[a]]

stratify takes as input a list of objects and a function f which for each object extracts a set of numbers and groups them such that:

  • the number of groups is minimal,
  • the groups are a partition of the input list,
  • given a group composed of object [o_1, ..., o_n] we have that the intersection of (f o_i) and (f o_j) is empty iff i =/= j,
  • if x precedes y in the input list then the group that contains x is the same or precedes the group that contains y.

stratify' :: Ord b => (a, [b]) -> StrataState a b -> StrataState a b

stratify' is the step the stratification algorithm makes for a single item.

stratum :: Ord b => [b] -> Map b Int -> Int

stratum returns the group to which the item must be added so that the stratify properties are satisfied.

strashow :: Show a => [[a]] -> String

strashow shows the result of the stratification algorithm.

make_tuples :: [[a]] -> [[a]]

make_tuples takes as input a list xs. each element x of xs represents a list of possibilities. make_tuples returns lists whose elements are chosen, in order, from each x for example make_tuples [[1, 2], [3, 4]] will return [[1, 3], [1, 4], [2, 3], [2, 4]] you can view the results as the possible "paths" through xs.

make_tuples [[a,b,c],[d,e,f],[g,h,i]]
a    /- d -\      g
    /       \
b -/    e    \    h
              \
c       f      \- i -> [b, d, i]
